Já ouviu falar em GNU?

Eis os 4 principais e fundamentais fundamentos do GNU:
1. Utilizar o software para qualquer fim.
2. Estudar o código do software.
3. Modificar o código do software.
4. Redistribuir cópias do software (sem que esses perca estes mesmo 4 princípios).
